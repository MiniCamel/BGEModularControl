## Installation

BGEModularControl is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
platform :ios, '9.0'

target '<#TargetName#>' do

pod 'BGEModularControl', '~> 1.0.0'

end
```
